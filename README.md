# Daily Backend Practice

> 每天 30 分钟，从前端工程师到 AI Agent 全栈工程师

## 学习计划

| 周 | 主题 | 目标 |
|---|------|------|
| 01 | HTTP & Express 基础 | 能从零搭一个 REST API |
| 02 | 数据库（SQL + ORM） | 能设计表结构、写 CRUD |
| 03 | 认证与鉴权 | Token/OAuth/API Key |
| 04 | 向量数据库 + Embedding | RAG 的核心后端组件 |
| 05 | 消息队列 & 事件系统 | Agent 异步任务编排 |
| 06 | Streaming & WebSocket | LLM 流式输出 |
| 07 | 缓存策略 | 面试必考 + LLM 成本优化 |
| 08 | 容器化 & 部署 | Docker + CI/CD |
| 09 | 系统设计综合 | 串联所有知识画架构 |

## 规则

1. **AI 当字典，不当代笔** — 查语法 OK，不能让 AI 直接生成完整方案
2. **每天一个闭环** — 看 TASK.md → 写代码到 solution/ → commit
3. **思考题必答** — 用自己的话，写在 solution/THINK.md 里
4. **不要求完美** — 能跑就行，重要的是你自己写的

## 目录结构

```
week-XX-主题/
├── README.md        # 本周概述
├── day-01/
│   ├── TASK.md      # 任务说明
│   └── solution/    # 你的代码
├── day-02/
│   ...
```
